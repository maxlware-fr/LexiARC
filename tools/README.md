# Outils LexiARC

Les fichiers python sont des outils permettant d'avoir des informations complémentaires sur les fichiers compilés.

## Desciption des fichiers :
- lexianalyzer.py : CLI python intégré permettant d'avoir toutes les informations de la console
- cpu.py : poser la structure minimale et correcte de l'émulateur
- disasm.py : Désassembleur Score7
- extract_target.py : extraction des premiers octets de Lead.sys
- fat32_generator.py : création d'une fausse Carte SD en FAT32 avec mtools
- build_sd_image.ps1 : Création de la carte SD (Ne marche pas)
